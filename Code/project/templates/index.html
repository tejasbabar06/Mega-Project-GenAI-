Multi-Agent Feature Engineering
Powered by LLM Agents

A Constrained-Topology Multi-Agent Framework for Planning-Based Feature Engineering — upload your dataset and watch three AI agents collaborate to preprocess and select the best features automatically.

🚀 Get Started ↓ Learn More

Agent Architecture

Three Agents · One Pipeline

🔍
DataAnalysisAgent
Identifies column types, detects missing values, and produces a structured analysis report.
→
🗺️
FeaturePlanningAgent
Decides the preprocessing pipeline — imputation, encoding, and scaling strategies — based on the analysis.
→
⭐
FeatureSelectionAgent
Executes the plan, then selects the most informative features via 7 ML methods including PCA, with visualizations.

What It Does

Capabilities

📊
Smart Data Analysis
Automatically classifies columns, counts missing values, and summarises the dataset structure.
⚙️
Automated Preprocessing
Mean / mode imputation, LabelEncoder for categoricals, and StandardScaler for numerics — applied automatically.
🎯
7 Selection Methods
SelectKBest, RandomForest, Lasso, GradientBoosting, ExtraTrees, Mutual Information, and PCA with importance scores.
📊
Visualizations & Logs
Missing values charts, feature importance bar charts, PCA variance plots, and detailed agent decision logs.
